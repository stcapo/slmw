# Phase 2 T3-3S PCB Review Checklists

> Updated: 2026-06-08  
> Use this file during PM/intern reviews. Do not advance to the next gate until every required item is checked or logged as an accepted risk.

## 1. Kickoff Checklist

- [ ] Intern has read `Phrase2/phase2_t3_3s_pcb_schedule.md`.
- [ ] Intern has opened `Phrase2/T3-3S_datasheet.pdf`.
- [ ] First board scope is confirmed as 12/24V low-voltage only.
- [ ] 220V AC, 48V, Zigbee, and miniaturization are explicitly out of scope for v0.1.
- [ ] Phase 1 common-anode CCT wiring is explained: red = common positive, yellow = WW negative, white = CW negative.
- [ ] Low-side MOSFET control is explained.
- [ ] Prototype-only status of YYNMOS-4/NMOS-4 is explained.
- [ ] PM and intern agree on deliverable names.

## 2. T3-3S Schematic Checklist

- [ ] T3-3S symbol pin numbers match the datasheet.
- [ ] Footprint is created from the T3-3S recommended footprint, not copied from WB3S.
- [ ] Pin 1 orientation is clearly marked on the schematic and PCB silkscreen.
- [ ] `VBAT/3V3` is connected only to regulated 3.3V.
- [ ] `GND` connects to board ground.
- [ ] `P24` is assigned to WW PWM.
- [ ] `P32` is assigned to CW PWM.
- [ ] `TX1` and `RX1` are available for debug/UART access.
- [ ] `CEN` is available as a reset/test point.
- [ ] `3V3`, `GND`, `P24`, `P32`, `TX1`, `RX1`, and `CEN` have accessible test points.
- [ ] Unused pins are either no-connect or assigned intentionally in the pin map.

## 3. Power Checklist

- [ ] Input connector is labeled `VIN+` and `VIN-`.
- [ ] Input voltage range for v0.1 is documented as 12/24V DC.
- [ ] 3.3V rail is generated by a Buck regulator suitable for 24V input.
- [ ] Buck input capacitors are placed close to the Buck input pins in the layout plan.
- [ ] Buck output capacitors are placed close to the Buck output path in the layout plan.
- [ ] Buck feedback network follows the selected regulator datasheet.
- [ ] 3.3V rail has local decoupling near T3-3S.
- [ ] No 12V/24V net can touch T3-3S GPIO, UART, CEN, or 3.3V net.
- [ ] Input reverse/polarity protection decision is documented in the BOM notes.
- [ ] Input transient protection decision is documented in the BOM notes.

## 4. MOSFET and LED Output Checklist

- [ ] Output connector is labeled `LED+`, `WW-`, and `CW-`.
- [ ] `LED+` is common positive from the DC input positive path.
- [ ] WW MOSFET switches the WW negative channel.
- [ ] CW MOSFET switches the CW negative channel.
- [ ] MOSFETs are logic-level devices specified for 3.3V gate drive.
- [ ] MOSFET voltage rating has margin above 24V.
- [ ] MOSFET current and thermal design are documented against the intended LED load.
- [ ] Each MOSFET gate has a series resistor.
- [ ] Each MOSFET gate has a pulldown resistor to ground.
- [ ] MOSFET source/drain orientation is checked against the package pinout.
- [ ] High-current traces are wider than signal traces and routed directly.
- [ ] LED current does not return through thin T3-3S signal ground traces.

## 5. Placement Checklist

- [ ] T3-3S is placed near a PCB edge.
- [ ] T3-3S antenna points outward.
- [ ] T3-3S antenna area has no copper pour.
- [ ] T3-3S antenna area has no traces or vias.
- [ ] T3-3S keep-out areas from the footprint are not used as pads or routing space.
- [ ] Buck inductor and switching node are away from the T3-3S antenna.
- [ ] MOSFETs are near the LED output connector.
- [ ] Buck is near the input connector but separated from the antenna.
- [ ] Test points are reachable by meter probes or logic analyzer clips.
- [ ] Connectors match the expected wiring direction in the physical lamp.

## 6. Routing Checklist

- [ ] VIN current loop is short and direct.
- [ ] LED high-current loop is short and direct.
- [ ] Buck switching loop is compact.
- [ ] Ground return path is continuous and sensible.
- [ ] T3-3S antenna keep-out remains clean after copper pour.
- [ ] PWM gate traces are not routed through noisy Buck switching areas.
- [ ] UART/test traces are accessible and not hidden under connectors.
- [ ] Thermal copper is provided for MOSFETs where the package supports it.
- [ ] Silkscreen labels remain readable after component placement.
- [ ] DRC is clean or all remaining DRC items are accepted in the issue log.
- [ ] ERC is clean or all remaining ERC items are accepted in the issue log.

## 7. Gerber Release Checklist

- [ ] Schematic PDF exported.
- [ ] PCB top screenshot exported.
- [ ] PCB bottom screenshot exported.
- [ ] 3D preview exported.
- [ ] BOM exported.
- [ ] Pick/place file exported.
- [ ] Gerber zip exported.
- [ ] Assembly drawing exported.
- [ ] DRC/ERC report exported.
- [ ] Issue log updated.
- [ ] PM approves release for fabrication.

## 8. Bring-Up Checklist

Before power:

- [ ] Visual inspection: no reversed T3-3S, Buck IC, diode, electrolytic capacitor, connector, or MOSFET.
- [ ] Visual inspection: no obvious solder bridge.
- [ ] Meter check: `VIN+` to `VIN-` is not shorted.
- [ ] Meter check: `3V3` to `GND` is not shorted.
- [ ] Meter check: `LED+` to `WW-` and `LED+` to `CW-` are not hard-shorted unless load is intentionally connected.

12V power:

- [ ] Power supply current limit is set before first power-up.
- [ ] 12V is connected to `VIN+` and `VIN-`.
- [ ] 3.3V output is measured.
- [ ] T3-3S current or board current is recorded.
- [ ] No smoke, smell, overheating, or supply current runaway.
- [ ] UART/debug access is checked if firmware/log path is ready.

PWM and load:

- [ ] `P24` PWM is measured unloaded.
- [ ] `P32` PWM is measured unloaded.
- [ ] WW single-channel LED test passes.
- [ ] CW single-channel LED test passes.
- [ ] Dual-channel CCT test passes.
- [ ] CCT direction is recorded as correct or reversed.

24V and thermal:

- [ ] 24V test begins only after 12V test passes.
- [ ] 3.3V output remains stable at 24V input.
- [ ] 24V WW single-channel test passes.
- [ ] 24V CW single-channel test passes.
- [ ] 24V dual-channel CCT test passes.
- [ ] 30-minute temperature record is completed for T3-3S, Buck, MOSFETs, input connector, and LED connector.

## 9. Acceptance Criteria

- [ ] No known schematic hard error remains.
- [ ] No known footprint/orientation hard error remains.
- [ ] 12V input works.
- [ ] 24V input works.
- [ ] 3.3V is stable in both cases.
- [ ] T3-3S can be powered and debugged.
- [ ] WW/CW PWM can be measured.
- [ ] WW/CW MOSFET channels switch the LED load independently.
- [ ] 30-minute thermal test has no abnormal heat.
- [ ] v0.2 issue list is written.
