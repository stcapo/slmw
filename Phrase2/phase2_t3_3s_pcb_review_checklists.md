# Phase 2 T3-3S PCB 评审检查清单

> 更新日期：2026-06-08
> 用途：项目负责人和实习生每次评审时逐项检查。没有检查通过，或没有在问题清单里明确接受风险，就不要进入下一阶段。

## 阅读说明

这份文件不是给工程师“自由发挥”的参考资料，而是每次评审时要逐项打勾的清单。产品经理可以重点看每项是否有明确证据，例如截图、PDF、BOM、测试记录或照片。

## 1. 启动会检查清单

- [ ] 实习生已阅读 `Phrase2/phase2_t3_3s_pcb_schedule.md`。
- [ ] 实习生已打开并熟悉 `Phrase2/T3-3S_datasheet.pdf` 规格书。
- [ ] 已确认首板只做 12/24V 低压版本。
- [ ] 已明确 v0.1 不做 220V AC、48V、Zigbee 和小型化优化。
- [ ] 已讲清 Phase 1 的共阳双色温接线：红线 = 公共正极，黄线 = 暖白负极，白线 = 冷白负极。
- [ ] 已讲清低边 MOSFET 控制方式：控制 LED 两路负极，不控制公共正极。
- [ ] 已讲清 YYNMOS-4/NMOS-4 只是 Phase 1 原型模块，不是最终 PCB 电路。
- [ ] 项目负责人和实习生已统一交付物命名。

## 2. T3-3S 原理图检查清单

- [ ] T3-3S 原理图符号的引脚编号与规格书一致。
- [ ] T3-3S PCB 封装按规格书推荐封装（recommended footprint）创建，没有复制 WB3S。
- [ ] Pin 1 方向在原理图和 PCB 丝印上都标清楚。
- [ ] `VBAT/3V3` 只连接到稳定的 3.3V 电源。
- [ ] `GND` 正确接到整板地。
- [ ] `P24` 分配给 WW 暖白 PWM。
- [ ] `P32` 分配给 CW 冷白 PWM。
- [ ] `TX1` 和 `RX1` 已引出，方便后续 UART / 调试。
- [ ] `CEN` 已引出，方便复位或测试。
- [ ] `3V3`、`GND`、`P24`、`P32`、`TX1`、`RX1`、`CEN` 都有可测测试点。
- [ ] 未使用引脚已明确标为不接，或已在引脚分配表里说明用途。

## 3. 电源检查清单

- [ ] 输入端子丝印清楚标为 `VIN+` 和 `VIN-`。
- [ ] v0.1 输入电压范围写清楚：只支持 12/24V DC。
- [ ] 3.3V 由适合 24V 输入的 Buck 降压电路产生。
- [ ] Buck 输入电容在 PCB 布局计划中靠近 Buck 输入脚。
- [ ] Buck 输出电容在 PCB 布局计划中靠近 Buck 输出路径。
- [ ] Buck 反馈电阻网络按所选 Buck 规格书连接。
- [ ] T3-3S 附近有 3.3V 去耦电容。
- [ ] 12V/24V 不会接到 T3-3S 的 GPIO、UART、CEN 或 3.3V。
- [ ] 输入防反接方案已在 BOM 备注里写清楚。
- [ ] 输入浪涌 / 瞬态保护方案已在 BOM 备注里写清楚。

## 4. MOSFET 和 LED 输出检查清单

- [ ] 输出端子丝印清楚标为 `LED+`、`WW-`、`CW-`。
- [ ] `LED+` 是从输入正极来的公共正极。
- [ ] WW MOSFET 控制暖白负极通道。
- [ ] CW MOSFET 控制冷白负极通道。
- [ ] MOSFET 是 3.3V 栅极信号可可靠驱动的逻辑电平器件。
- [ ] MOSFET 耐压高于 24V，并留有余量。
- [ ] MOSFET 电流和散热能力已按目标 LED 负载做过说明。
- [ ] 每个 MOSFET 栅极都有串联电阻。
- [ ] 每个 MOSFET 栅极都有下拉电阻接地，避免上电时误亮。
- [ ] MOSFET 源极 / 漏极方向已按封装管脚定义检查。
- [ ] 大电流走线比普通信号线更宽，且路径直接。
- [ ] LED 大电流不会从 T3-3S 附近的细信号地线返回。

## 5. 器件摆放检查清单

- [ ] T3-3S 放在 PCB 边缘附近。
- [ ] T3-3S 天线朝外。
- [ ] T3-3S 天线区域没有覆铜。
- [ ] T3-3S 天线区域没有走线或过孔。
- [ ] T3-3S 封装中的禁布区（keep-out）没有被当作焊盘或走线区域。
- [ ] Buck 电感和开关节点远离 T3-3S 天线。
- [ ] MOSFET 靠近 LED 输出端子。
- [ ] Buck 靠近输入端子，但与天线区域保持距离。
- [ ] 测试点能被万用表表笔或逻辑分析仪夹子碰到。
- [ ] 端子位置符合真实灯具线束方向，避免装配时绕线或接反。

## 6. 走线检查清单

- [ ] VIN 输入大电流回路短而直接。
- [ ] LED 输出大电流回路短而直接。
- [ ] Buck 开关回路紧凑。
- [ ] 地线回流路径连续、清楚，不绕远路。
- [ ] 覆铜后 T3-3S 天线禁布区仍然干净。
- [ ] PWM 栅极线没有穿过 Buck 开关噪声区域。
- [ ] UART / 测试线可触达，没有被压在端子下面。
- [ ] MOSFET 封装支持散热铜皮的位置已铺铜。
- [ ] 器件摆放后丝印仍然能看清。
- [ ] DRC 无错误；如有剩余项，已在问题清单中接受风险。
- [ ] ERC 无错误；如有剩余项，已在问题清单中接受风险。

## 7. Gerber 发布检查清单

- [ ] 已导出 PCB 正面截图。
- [ ] 已导出 PCB 反面截图。
- [ ] 已导出 3D 预览图。
- [ ] 已导出 BOM。
- [ ] 已导出贴片坐标文件。
- [ ] 已导出 Gerber zip。
- [ ] 已导出装配图。
- [ ] 已导出 DRC/ERC 报告。
- [ ] 已更新问题清单。
- [ ] 项目负责人已批准发板打样。

## 8. 上电测试检查清单

上电前：

- [ ] 目检确认 T3-3S、Buck IC、二极管、电解电容、端子、MOSFET 没有装反。
- [ ] 目检确认没有明显焊桥。
- [ ] 万用表确认 `VIN+` 到 `VIN-` 没有短路。
- [ ] 万用表确认 `3V3` 到 `GND` 没有短路。
- [ ] 没接负载时，确认 `LED+` 到 `WW-`、`LED+` 到 `CW-` 没有硬短路。

12V 上电：

- [ ] 第一次上电前，电源已设置限流。
- [ ] 12V 正负已正确接到 `VIN+` 和 `VIN-`。
- [ ] 已测量 3.3V 输出。
- [ ] 已记录 T3-3S 电流或整板输入电流。
- [ ] 没有冒烟、异味、过热或电源电流失控。
- [ ] 如果固件 / 日志路径已准备好，已检查 UART / 调试入口。

PWM 和负载：

- [ ] 空载测到 `P24` PWM。
- [ ] 空载测到 `P32` PWM。
- [ ] WW 单路 LED 测试通过。
- [ ] CW 单路 LED 测试通过。
- [ ] 双路 CCT 混光测试通过。
- [ ] 已记录色温方向是正确还是相反。

24V 和温升：

- [ ] 只有 12V 测试通过后，才开始 24V 测试。
- [ ] 24V 输入时，3.3V 输出仍稳定。
- [ ] 24V WW 单路测试通过。
- [ ] 24V CW 单路测试通过。
- [ ] 24V 双路 CCT 测试通过。
- [ ] 已完成 30 分钟温度记录，位置包括 T3-3S、Buck、MOSFET、输入端子和 LED 端子。

## 9. 首板验收标准

- [ ] 没有已知原理图硬伤。
- [ ] 没有已知封装或方向硬伤。
- [ ] 12V 输入工作正常。
- [ ] 24V 输入工作正常。
- [ ] 两种输入下 3.3V 都稳定。
- [ ] T3-3S 能上电并可调试。
- [ ] WW/CW PWM 能测到。
- [ ] WW/CW MOSFET 能独立驱动 LED 负载。
- [ ] 30 分钟温升测试没有异常发热。
- [ ] v0.2 问题清单已写清楚。
