# 工作日报：Phase 1 低压硬件闭环完成

> 日期：2026-05-27  
> 项目：通用智能灯具中间层解决方案  
> 阶段：Phase 1 低压 DC 原型验证  
> 当前状态：T5AI-Core 到 MOSFET 到 12V LED 的单路和双路双色温硬件闭环均已通过；下一步进入 Tuya App 配网、开关、亮度、色温控制验证。

## 一、今日目标

今天目标是把 Phase 1 从“PWM 空载与电源/MOSFET 准备完成”推进到“真实 12V LED 负载闭环通过”，重点验证：

- `T5AI-Core P24` 能否通过 `YYNMOS-4 PWM1` 控制 12V 单色灯带。
- `T5AI-Core P24/P25` 能否分别通过 `YYNMOS-4 PWM1/PWM2` 控制双色温灯带的暖白/冷白通道。
- 双色温灯带线序、共阳结构和 MOSFET 低边开关接法是否成立。

## 二、今日完成工作

### 1. Step 10：单路 12V 灯带闭环验证通过

在 T5AI-Core 已烧录 `firmware/t5_pwm_smoke_test/t5_pwm_smoke_test.ino` 的基础上，完成单路接线：

```text
T5 P24 -> YYNMOS-4 PWM1
T5 GND -> YYNMOS-4 GND1
12V+ -> YYNMOS-4 DC+
12V- -> YYNMOS-4 DC-
12V 单色灯带 + -> YYNMOS-4 OUT1+
12V 单色灯带 - -> YYNMOS-4 OUT1-
```

实测现象：

```text
灯带每 3 秒按 暗 -> 中 -> 亮 -> 灭 -> 全亮 变化
```

结论：`T5AI-Core -> PWM -> YYNMOS-4 -> 12V 单色 LED` 单路低压闭环通过。

### 2. 双色温灯带线序确认

对 12V/24V 双色温灯带进行线序实测：

```text
红线接 12V 正极
黄线 WW 接 12V 负极 -> 亮暖白
白线 CW 接 12V 负极 -> 亮冷白
```

结论：

```text
红线 = 公共正极
黄线 = WW 暖白负极通道
白线 = CW 冷白负极通道
```

该灯带为共阳极结构，适合用 YYNMOS-4 这种低边 MOSFET 模块控制各路负极通道。

### 3. Step 11：双路双色温硬件闭环验证通过

完成双路接线：

```text
T5 P24 -> YYNMOS-4 PWM1 -> 黄线 WW 暖白负极
T5 P25 -> YYNMOS-4 PWM2 -> 白线 CW 冷白负极
T5 GND -> YYNMOS-4 GND1/GND2
12V+ -> YYNMOS-4 DC+ / 灯带红线公共正极
12V- -> YYNMOS-4 DC-
```

先用原五档程序测试：

```text
P24: 10% -> 50% -> 90% -> 0% -> 100%
P25: 90% -> 50% -> 10% -> 100% -> 0%
```

观察到灯光存在周期性变化，但肉眼对偏冷、中性、偏暖的区分不够明显。因此新增极限冷暖两档测试程序进行复核。

### 4. 新增极限冷暖两档测试程序

新增文件：

```text
firmware/t5_cct_extreme_test/t5_cct_extreme_test.ino
```

程序逻辑：

```text
PURE_COLD:
  WW P24 duty = 0%
  CW P25 duty = 100%

PURE_WARM:
  WW P24 duty = 100%
  CW P25 duty = 0%
```

实测结果：

```text
极限冷白和极限暖白区别明确
双路 MOSFET 控制有效
P24/P25 到双色温灯带两路负极通道的硬件闭环成立
```

结论：Step 11 双路双色温硬件闭环通过。

## 三、今日问题与处理

| 问题 | 判断 | 处理结果 |
| --- | --- | --- |
| 剥线钳、剪钳仍未到货 | 剪刀可临时代替，但线头可靠性较差 | 今天完成验证；后续仍建议补购工具 |
| 五档混光肉眼不够明显 | 总占空比恒定为 100%，只变色温，肉眼区分有限 | 新增极限冷/暖两档程序复核 |
| 冷白通道肉眼可见闪烁 | 当前 PWM 频率为 200Hz，原型 MOSFET 模块标称上限 500Hz | 记录为原型频率风险；产品化阶段目标提高到 2kHz 以上 |
| 通道方向后续可能反 | P24/P25 与 WW/CW 的映射可能影响 App 色温滑条方向 | 后续 App 控制阶段通过映射或接线调整修正 |


- 当前 YYNMOS-4/NMOS-4 只是原型模块，不作为量产功率级。
