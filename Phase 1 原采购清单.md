# Phase 1 采购清单与可行性校正版

> 版本：2026-05-21 T5AI-Core 最终下单版  
> 范围：第一阶段只做低压 DC 原型验证，禁止在面包板/飞线环境接入 220V 市电。

## 一、第一阶段结论

Phase 1 不再围绕旧的 T3-3S 裸模组、T2-U、WB3S/NANO 等入口反复选型。当前主路径已经固定：

```text
T5AI-Core
  -> 3.3V PWM 信号
  -> YYNMOS-4 / NMOS-4 (3-5V) MOSFET 模块
  -> 12V 恒压双色温/三色 LED 灯带
  -> 12V2A 成品隔离电源
```

第一阶段要验证三件事：

1. `T5AI-Core` 可以稳定运行并输出两路可控 PWM。
2. `YYNMOS-4 / NMOS-4 (3-5V)` 可以被 T5 的 3.3V PWM 驱动，并控制 LED 负载亮灭/调光。
3. 低压 LED 负载、MOSFET 模块和电源在连续运行时温升可控。

## 三、第一阶段接线原则

```text
12V 成品电源
  + -> YNMOS-4 电源正极
  - -> YNMOS-4 电源负极

T5AI-Core PWM1 -> YNMOS-4 PWM1
T5AI-Core GND  -> YNMOS-4 GND1
T5AI-Core PWM2 -> YNMOS-4 PWM2
T5AI-Core GND  -> YNMOS-4 GND2

YNMOS-4 输出1 -> LED 暖白/一路负载
YNMOS-4 输出2 -> LED 冷白/另一路负载
```

到货后先看灯带焊盘或线标。如果不是公共正极三线/多线结构，不要直接按上图接。

## 四、第一阶段验收标准

| 项目 | 通过标准 |
| --- | --- |
| T5 基础运行 | Type-C 供电正常，串口/日志可观察，示例程序可运行 |
| PWM 输出 | 逻辑分析仪能测到至少两路 PWM，占空比随程序或 App 控制变化 |
| MOSFET 控制 | `YYNMOS-4 / NMOS-4 (3-5V)` 可被 T5 3.3V PWM 驱动，LED 亮度可变化 |
| 12V 负载 | 12V 灯带可完成开关、亮度、两路混光或至少双路独立调光 |
| 温升 | 连续运行 30 分钟，MOSFET 模块、Buck、灯带温度记录完整，无异常烫手/异味 |
| 供电 | 12V 电源稳定；LM2596 输出接入任何主板前必须先用万用表确认电压 |

由于当前 MOSFET 模块标称最大 PWM 频率为 500Hz，第一阶段不再用 `>=2kHz` 作为 YYNMOS/NMOS 模块验收线。产品化阶段仍需重新设计更高频、更小型的 MOSFET 功率级。

## 五、暂不进入本轮默认验证

| 原物料/方案 | 当前处理 | 原因 |
| --- | --- | --- |
| T3-3S/T2-U 裸模组主路径 | 后移 | 第一阶段已固定 T5AI-Core，避免继续卡在焊接/转接板 |
| 48V LED 功率级 | 后移 | 本次没有 48V 电源、48V MOSFET 和真正宽压 Buck |
| XL7015E1 裸 Buck 芯片 | 后移到 PCB 阶段 | 需要外围器件和 PCB 布局，不适合小白第一轮飞线 |
| AO3400A 直接飞线 | 不作为首轮主方案 | 裸 SOT-23 手工接线风险高，先用 YYNMOS/NMOS 模块 |
| 24V 上电测试 | 条件满足后再做 | 订单里有 24V 负载，但截图未显示独立 24V 电源适配器 |

## 六、采购优先级

1. 本轮按截图订单直接购买，不再继续扩展候选清单。
2. 到货后先跑 `T5AI-Core + YYNMOS-4/NMOS-4 + 12V 灯带 + 12V2A 电源`。
3. 12V 闭环跑通后，再使用万用表、逻辑分析仪、温度计记录 PWM、供电和温升。
4. 24V 与 48V 不抢跑；24V 等电源条件明确后再测，48V 进入 Phase 2。
