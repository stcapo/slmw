# 工作日报：Phase 1 原型实施启动

> 日期：2026-05-25  
> 项目：通用智能灯具中间层解决方案  
> 阶段：Phase 1 低压 DC 原型验证  
> 当前状态：T5AI-Core 单板、串口、烧录、PWM 输出、12V 电源均完成基础可用性验证；等待 DC 转接头与鳄鱼夹到货后继续 MOSFET/灯带接线。

## 一、今日目标

今天主要目标是把第一批采购物料从“到货状态”推进到“可实施状态”，完成基础开箱、上电、安全检查、开发环境安装和首轮硬件可用性验证，为后续 `T5AI-Core -> PWM -> MOSFET -> 12V LED` 闭环测试做准备。

## 二、今日完成工作

### 1. 物料开箱与基础核对

已拆封并初步确认以下关键物料：

- `T5AI-Core` 开发板
- `YYNMOS-4 / NMOS-4 (3-5V)` 四路 MOSFET 模块
- `12V2A` 成品电源适配器
- `Kingst LA1010` 逻辑分析仪
- 万用表
- 12V LED 负载相关物料

同时确认当前阶段仍坚持低压 DC 验证，不接入 220V 市电，不直接连接裸 AC-DC 模块。

### 2. T5AI-Core 单板上电与基础排查

对 T5AI-Core 进行了 Type-C 单板上电测试。初始阶段曾出现 `5V/3.3V` 测点电压接近 0V、按 `RESET` 无日志的现象。

排查后确认原因是开发板电源开关未打开。打开电源开关后：

- 板载电源恢复正常。
- `5V` 和 `3.3V` 测点电压正常。
- 按 `RESET` 可看到启动日志。

结论：T5AI-Core 硬件上电正常，第一步单板可用性验证通过。

### 3. 串口识别与端口分工确认

macOS 下插入 T5AI-Core 后识别到两个串口：

```text
/dev/cu.usbmodem5AAE1666551
/dev/cu.usbmodem5AAE1666553
```

经 TuyaOpen Tools 和 Arduino 串口测试确认：

```text
/dev/cu.usbmodem5AAE1666551 -> 烧录/下载口
/dev/cu.usbmodem5AAE1666553 -> 调试日志口
```

该结论为后续烧录和日志观察提供了明确依据。

### 4. 开发环境安装与配置

完成 Arduino IDE 2 的基础使用确认，并整理了开发板管理器配置方式：

- Arduino 本地编译和烧录不需要登录账号。
- 添加 Tuya Open 开发板管理器地址。
- 安装 `Tuya Open` 开发板包。
- 选择 `TUYA_T5AI_CORE` 开发板。

同时解决了开发板选择过程中找不到 T5AI 板型的问题，明确原因是 Tuya Open 开发板包未安装。

### 5. PWM 冒烟测试程序编写与烧录

新增并烧录了 PWM 空载测试程序：

```text
firmware/t5_pwm_smoke_test/t5_pwm_smoke_test.ino
```

程序设计目标：

- 使用 T5AI-Core 输出两路 PWM 信号。
- `P24` 作为 `PWM_A`。
- `P25` 作为 `PWM_B`。
- 频率设置为 `200Hz`，低于当前 YYNMOS/NMOS 模块标称 `500Hz` 上限。
- 占空比每 3 秒变化一次，便于逻辑分析仪观察。

烧录结果显示：

```text
Write flash success
CRC check success
Reboot done
Flash write success
```

结论：自定义测试程序已成功写入 T5AI-Core。

### 6. PWM 输出验证

通过 Arduino Serial Monitor 观察到测试程序输出：

```text
PWM_A P24 duty=90%, PWM_B P25 duty=10%, freq=200Hz
PWM_A P24 duty=0%, PWM_B P25 duty=100%, freq=200Hz
PWM_A P24 duty=10%, PWM_B P25 duty=90%, freq=200Hz
```

通过 LA1010 逻辑分析仪已抓到周期性数字波形，说明 T5AI-Core 的 PWM 输出已经开始工作。

当前结论：Phase 1 的 `T5AI-Core PWM 空载单测` 已基本通过，后续只需在正式记录中补充频率、占空比截图或测量数值。

### 7. 12V2A 电源适配器单独验证

对 12V2A 成品适配器进行单独空载测量：

```text
输出形式：圆形 DC 插头
实测电压：12.25V
极性：内芯正极，外圈负极
状态：无短路、无异味、无异常发热
```

结论：12V2A 适配器单测通过，可作为后续 MOSFET 模块和 12V LED 负载供电来源。

### 8. MOSFET 模块端子确认

对 `YYNMOS-4 / NMOS-4` 模块进行正反面拍照检查，已确认背面丝印端子：

控制侧：

```text
PWM1 / GND1
PWM2 / GND2
PWM3 / GND3
PWM4 / GND4
```

功率侧：

```text
DC+ / DC-
OUT1+ / OUT1-
OUT2+ / OUT2-
OUT3+ / OUT3-
OUT4+ / OUT4-
```

后续第一轮建议只使用：

```text
PWM1 / GND1
DC+ / DC-
OUT1+ / OUT1-
```

### 9. 安全转接方案调整

原计划曾考虑剪开 12V 适配器低压输出线，但综合安全性、可逆性和原型调试便利性，决定暂不剪线。

后续补充采购：

- `5.5*2.5mm DC 母座转螺丝端子`
- `5.5*2.5mm DC 母座转红黑线`
- 鳄鱼夹测试线

等待转接物料到货后，再继续进行 MOSFET 模块和 12V LED 负载接线。

## 三、今日问题与处理

## 六、风险提醒

- 12V 电源不得接入 T5AI-Core 的 `5V/3V3/GPIO`。
- MOSFET 模块功率侧和 T5 控制侧必须共地，即 `T5 GND -> 模块 GND1`。
- 当前 YYNMOS/NMOS 模块标称 PWM 上限为 `500Hz`，第一阶段测试频率先固定在 `200Hz`。
- 后续接线前必须断开 12V 适配器电源，完成复查后再上电。
