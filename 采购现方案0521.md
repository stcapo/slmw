## 一、当前拍板结论

第一阶段目标收束为一件事：用 `T5AI-Core -> PWM -> MOSFET 模块 -> 12V/24V 恒压 LED 负载` 跑通 App/固件/调光闭环。

| 模块         | 当前决定                           | 边界                                      |
| ---------- | ------------------------------ | --------------------------------------- |
| 主开发板       | `T5AI-Core`                    | 已确定，作为第一阶段唯一主开发入口                       |
| MOSFET 功率级 | `YYNMOS-4 / NMOS-4 (3-5V)`，2 块 | 原型验证可用；标称最大 PWM 频率 500Hz，不作为量产最终电路      |
| LED 负载     | 12V/24V 灯带、灯片、涂鸦成品控制器辅助验证      | 到货后确认线序，优先用 12V 负载先跑通                   |
| 3.3V Buck  | LM2596 模块若干                    | 只用于 12V/24V 到 3.3V 的台架验证；不覆盖 48V 宽压产品目标 |
| 测试工具       | 万用表、逻辑分析仪、热电偶温度计、USB-TTL       | 一次配齐，降低后续排障成本                           |

旧口径处理：

- `T3-3S`、`T2-U`、`WB3S/NiceMCU` 不再作为第一阶段主路径；NiceMCU 仅保留为便宜备板/对照实验。
- `HX-D012` 不再作为本轮第一优先采购；本轮用已选的涂鸦五合一灯带控制器做成品控制器对照。
- `XL7015E1`、宽压 48V Buck、48V MOSFET 功率级后移到 Phase 2/PCB 阶段。

## 三、到货后执行顺序

1. 先只给 `T5AI-Core` 接 Type-C，跑通官方示例、串口日志、基础 GPIO/PWM 输出。
2. 用万用表单独测试 `12V2A 电源`，确认空载约 12V。
3. 用 `12V 电源 -> YYNMOS-4/NMOS-4 -> 12V 双色温灯带` 跑通两路 PWM 调光；先不用 Buck 给 T5 供电。
4. 用 `LA1010` 逻辑分析仪测 T5 的两路 PWM 频率和占空比，确认 App/程序调节时信号确实变化。
5. 用 `TA610A` 记录 MOSFET 模块、灯带、Buck 模块连续运行 30 分钟温度。
6. `LM2596` 模块先单独接 12V 输入，输出确认 3.3V 后，再用假负载/小负载测试；不要一上来接 T5。
7. 12V 闭环稳定后，再做 24V 灯带/灯片测试；若没有安全可靠的 24V 电源，本轮 24V 先暂缓。

## 四、接线原则

第一轮 12V 灯带验证：

```text
12V 电源正极 -> YNMOS-4 电源正极
12V 电源负极 -> YNMOS-4 电源负极

T5 PWM1 -> YNMOS-4 PWM1
T5 GND  -> YNMOS-4 GND1
T5 PWM2 -> YNMOS-4 PWM2
T5 GND  -> YNMOS-4 GND2

YNMOS-4 输出1 -> 12V 灯带暖白一路
YNMOS-4 输出2 -> 12V 灯带冷白一路
```

到货后必须先确认灯带线序。如果灯带不是公共正极三线/多线结构，不要硬接，先重新画接线图。

## 五、关键边界

| 项目           | 当前处理                                                                            |
| ------------ | ------------------------------------------------------------------------------- |
| PWM 频率       | `YYNMOS-4 / NMOS-4` 标称最大 500Hz，所以第一阶段验收改为“能调光、能测到占空比变化、无肉眼明显闪烁”。最终产品仍建议 >=2kHz。 |
| 48V 宽压       | 本次订单没有真正覆盖 48V 宽压 Buck，也没有 48V 功率 MOSFET。48V 后移。                                |
| 24V 电源       | 订单里有 24V 灯带/灯片，但截图未看到独立 24V 电源适配器。没有 24V 电源前，不做 24V 上电测试。                       |
| LM2596       | 适合 12V/24V 台架验证，不是 12-48V 产品化 Buck。                                             |
| AO3400A      | 可作为后续 PCB 预研料；裸 SOT-23 不作为小白第一阶段主接线方案。                                          |
| NiceMCU WB3S | 只作备板/对照实验，不再改变主路线。                                                              |
