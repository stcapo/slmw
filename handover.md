## 一、项目背景（一句话）

打造一个模块化"智能中间层"，让传统灯具无需改动驱动器即可接入涂鸦 App，实现 WiFi/Zigbee 控制开关、亮度、色温。目标市场：海外非标、异形造型灯具。

### ✅ 已完成步骤

| 步骤 | 内容 | 关键结果 |

| Step 5 | 烧录 PWM 冒烟测试程序 | Arduino IDE + TUYA_T5AI_CORE，写入成功，CRC 通过 |
| Step 6 | PWM 输出确认 | P24/P25 输出 200Hz 方波，占空比随程序 3 秒循环变化 |
| Step 7 | LA1010 逻辑分析仪抓波形 | 两路周期性方波已确认 |
| Step 8 | 12V 电源单独测量 | 实测 12.25V，内正外负，通过 |
| Step 9 | MOSFET 端子识别 | 丝印确认：PWM1~4 / GND1~4 / DC+ DC- / OUT1~4 ± |
| Step 10 | T5 → MOSFET → 12V 单色灯带闭环 | P24 → PWM1 成功驱动单色暖白灯带，亮度每 3 秒按 10%→50%→90%→0%→100% 变化 |
| Step 11 | P24/P25 双路双色温混光 | 双色温灯带共阳线序确认；双路 MOSFET 控制有变化；极限冷/暖两档测试通过 |

### 🔄 当前主线

**Step 11 已通过：T5AI-Core P24/P25 PWM → YYNMOS-4 PWM1/PWM2 → 12V 双色温灯带双路硬件闭环验证成功。**

关键结果：
- T5AI-Core 重新 USB 上电后，串口持续输出 `PWM_A P24 duty=...` 日志
- 12V 单色暖白灯带按预期每 3 秒经历：暗 → 中 → 亮 → 灭 → 全亮
- 说明控制侧共地、PWM1 输入、MOSFET 低边开关、12V 功率侧和 LED 极性均已跑通
- 12V/24V 双色温灯带线序已确认：红线为公共正极，黄线 WW 接负极亮暖白，白线 CW 接负极亮冷白
- `firmware/t5_cct_extreme_test/t5_cct_extreme_test.ino` 极限测试通过，冷白/暖白两档区别明确
- 冷白通道在 200Hz 下肉眼可感知闪烁，记录为原型频率风险；YYNMOS-4 标称 500Hz 上限内先不强行升频，产品化功率级后续目标应提升到 2kHz 以上

## 七、关键边界与禁止事项

| 事项 | 说明 |
|------|------|
| **12V 绝不接 T5** | T5 用 Type-C 供电，12V 只进 MOSFET 模块和灯带 |
| **不碰 220V** | 第一阶段全程低压 DC，不接市电 |
| YYNMOS-4 频率上限 | 标称 500Hz，程序设为 200Hz，不作为量产最终电路 |
| LM2596 Buck | 只做 12V/24V 台架验证，不代表 48V 宽压方案完成 |
| 24V 测试前提 | 必须有可靠 24V 电源，不能用当前 18.5V 测 12V 负载 |
| 5-in-1 成品控制器 | 接负载后无响应，疑似损坏，已暂停不再折腾 |
