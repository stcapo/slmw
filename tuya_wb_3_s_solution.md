# 涂鸦智能照明中间层方案（2026-05-21 T5AI-Core 校正版）

基于“低压 DC 供电 -> 智能中间层 -> 冷暖/三色 LED 负载”的架构，本文同步当前第一阶段最终采购口径。

## 1. 当前系统定位

```text
12V/24V 成品隔离电源
  -> 智能中间层原型：T5AI-Core + YYNMOS-4/NMOS-4 MOSFET 模块
  -> 12V/24V 恒压 LED 灯带/灯片
```

关键边界：

- 第一阶段不直接接触 220V 市电。
- 第一阶段主开发板固定为 `T5AI-Core`，不再继续比较 T3-3S/T2-U/WB3S/NANO。
- `YYNMOS-4 / NMOS-4 (3-5V)` 只作为原型 MOSFET 功率模块，标称最大 PWM 频率 500Hz，不作为最终量产电路。
- 本次 LM2596 模块只覆盖 12V/24V 到 3.3V 的台架验证，不覆盖 48V 宽压产品目标。

## 4. Phase 1 实施步骤

1. T5 单板上电：只用 Type-C，确认开发板运行、日志和示例程序。
2. PWM 单测：用 LA1010 测两路 PWM，占空比变化可见。
3. 12V 负载单测：确认 12V2A 电源输出，再接 YYNMOS-4/NMOS-4 和 12V 灯带。
4. 双路调光：T5 PWM1/PWM2 分别控制两路 LED，观察亮度和混光效果。
5. 温升记录：连续运行 30 分钟，记录 MOSFET 模块、灯带和 Buck 温度。
6. Buck 单测：LM2596 模块先空载/小负载测输出，确认 3.3V 后再考虑接入主控供电实验。
7. 24V 后测：只有在 24V 电源条件明确时才接 24V 灯带/灯片。

## 5. Phase 1 验收

| 项目 | 通过标准 |
| --- | --- |
| T5 运行 | Type-C 供电正常，固件/示例可运行 |
| PWM | 两路 PWM 可测，占空比可调 |
| 12V LED 控制 | 至少两路 LED 可独立调光 |
| 调光体验 | 肉眼无明显异常跳变；若有闪烁，记录对应 PWM 频率 |
| 温升 | 连续 30 分钟无异常过热，温度记录完整 |
| 供电安全 | 所有 Buck 输出接主板前先用万用表确认 |

## 6. 下一步建议

第一轮只追求把闭环跑通。闭环稳定后，再进入 Phase 2：

- 重新选择产品化 MOSFET 功率级，目标 PWM 频率 >=2kHz。
- 重新选择真正覆盖 12V-48V 的宽压 Buck。
- 在 T5-E1/T3-3S 等模组之间做最终小 PCB 选型。
- 用真实 12V/24V 灯具驱动和灯带建立兼容性矩阵。
