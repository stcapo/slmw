## 一、项目愿景与定位

**目标：** 打造一个高度模块化、易于集成的“智能中间层”解决方案，用于快速实现传统灯具（尤其是海外非标、异形造型灯具）的智能化改造。

**核心价值：**

1. **解耦供电与智能：** 传统灯具驱动器仅负责供电，智能功能完全由独立中间层控制。
2. **降低改造门槛：** 无需改动原有驱动器或灯珠，实现“即插即智”。
3. **快速市场响应：** 依托成熟的涂鸦智能模块，快速开发、验证并量产，填补个性化智能灯具的市场空白。

## 二、核心技术架构

**架构图示：**

`市电 (220V)` → `成品认证AC-DC恒压电源/传统驱动兼容测试` → **`智能中间层（模块化PCB：含涂鸦模组、宽压3.3V Buck、PWM功率驱动）`** → `恒压冷暖LED灯板`

**技术要点：**

* **通信：**
  * **WiFi直连 (首选)：** 第一阶段使用 T5AI-Core 开发板快速跑通 App/固件/PWM/灯带闭环；Phase 2 Wi-Fi 首板已选定 T3-3S，用于小 PCB 产品化迁移。
  * **Zigbee组网：** 涂鸦ZS3L/ZS3L-IPEX (高端) 或 ZT3L (低成本)，配合涂鸦网关，实现全屋联动、本地控制、边缘计算。
* **控制：**
  * **独立PWM输出：** 涂鸦模组提供多路硬件PWM，独立驱动冷暖LED灯珠。
  * **智能中间层：** 负责开关、亮度、色温调节，完全脱离原驱动器。
* **供电：**
  * **通用3.3V电源转换：** 设计一个能适配市面多数传统驱动器输出直流电压（如12V-48V）的宽压Buck降压模块，为涂鸦模组提供稳定的3.3V供电。
* **适配性：**
  * **尺寸：** 模组小巧，易内嵌至异形花灯、面板灯、筒灯、吸顶灯、LED灯泡等。
  * **接口：** 模块化设计，方便与各种驱动器和LED连接。

## 三、硬件选型（涂鸦模块核心）

**1. WiFi直连方案 (用户零门槛，零售/个性化市场)：**
    *   **Phase 1 主推：** **T5AI-Core 开发板**
    *   **Phase 2 首板：** **T3-3S** 涂鸦 WiFi/BLE 模组。
    *   **理由：** Phase 1 已跑通 `App/固件 -> PWM -> MOSFET -> LED` 的原型链路；Phase 2 需要迁移到小 PCB，T3-3S 提供 5 路硬件 PWM、板载 PCB 天线和适合 LED 控制的资源，适合作为 Wi-Fi 主线首板。

**2. Zigbee + 网关方案 (全屋智能，商照，边缘计算)：**
    *   **高端稳定：** **涂鸦 ZS3L** (板载天线) / **ZS3L-IPEX** (外接天线)
        *   **理由：** 支持Zigbee 3.0，适合构建本地化、智能化的全屋系统。
    *   **低成本量产：** **涂鸦 ZT3L**
        *   **理由：** 功能同ZS3L，成本更低，适合标准化、大批量产品。

**核心配套硬件 (需自行设计PCB集成)：**

* **通用3.3V降压模块：** 第一阶段 LM2596 模块仅用于台架验证；Phase 2 首板先覆盖 12/24V 输入到 3.3V Buck，48V 宽压版本后移。
* **MOSFET驱动电路：** 根据LED功率选择，用于PWM信号放大。
* **连接器、PCB板、外壳。**

## 四、前期实施计划

**阶段一：核心功能验证 (2-3周)**

1. **目标：** 搭建最小原型，验证“低压恒压供电→3.3V Buck→Tuya WiFi/BLE模组→PWM驱动恒压LED”的核心流程。
2. **关键动作：**
   * ~~采购~~ **物料已到货**：T5AI-Core、YYNMOS-4/NMOS-4 (3-5V) MOSFET 模块、12V 恒压灯带、12V2A 成品电源均已到手；剥线钳、绝缘胶带、热缩管、打火机/热风枪、镊子、剪钳尚需补购。
   * 手工搭建电路；当前单路 `T5AI-Core P24 -> YYNMOS-4 PWM1 -> 12V 单色灯带` 和双路 `P24/P25 -> YYNMOS-4 PWM1/PWM2 -> 12V 双色温灯带` 均已验证通过，下一步推进 App 配网、开关、亮度、色温控制。
   * **重点：** 验证PWM精度、色温混合效果、3.3V电源的稳定性与电压兼容性。

**阶段二：模块化设计与试制 (3-4周)**

1. **目标：** 设计 T3-3S 低压“智能中间层”PCB 首板，实现 12/24V 双路双色温 CCT 功能样板。

   * **PCB设计：** 按 `Phrase2/phase2_t3_3s_pcb_schedule.md` 执行，首板固定为 T3-3S；PWM 映射为 `P24 -> WW`、`P32 -> CW`；预留 `3V3/GND/TX1/RX1/CEN/P24/P32` 测试点。

## 五、风险与挑战

* **3.3V电源兼容性：** 市面驱动器输出电压差异大，设计通用且高效的降压方案是关键。
* **热管理：** 模块需在灯具内部有限空间内散热，设计需要充分考虑。
* **EMC/EMI：** 跨国认证要求高，后期需进行专业测试和优化。
* **成本控制：** 在满足功能的前提下，持续优化BOM成本。
